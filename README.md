Quiz App is a web app for practicing multiple-choice quizzes built with React + Vite. 
The app allows users to take quizzes from flashcards, calculate results, save quiz history, and view the leaderboard. It has a simple, user-friendly interface, and data is managed through MockAPI and React Query.
